# はじめに

このサイトでは [Planning Poker] で構築されたコンポーネントのカタログを閲覧することができます。

# 使い方

左メニューからメニューから閲覧したいコンポーネントを選択してください。

# 開発

以下のルールで `[コンポーネント名].stories.ts` ファイルを実装します。

- 作成対象の `[コンポーネント名].svelte` ファイルと同じフォルダに、同じコンポーネント名で作成する。
- Storybook の[Guides](https://storybook.js.org/docs/svelte/get-started/install) や [Tutorials](https://storybook.js.org/docs/svelte/get-started/install)を参考にして実装する。

[Planning Poker]: https://github.com/redfe/planning-poker-svelte
[Storybook]: https://storybook.js.org/
