# Introduction

On this site, you can browse a catalog of components built with [Planning Poker].

# How to Use

Select the component you want to view from the left menu.

# Development

Implement the [component-name].stories.ts file using the following rules:

- Create the file in the same folder as [component-name].svelte.
- Use the same name as the component for the file name.
- Refer to [Storybook]'s [Guides](https://storybook.js.org/docs/svelte/get-started/install) and [Tutorials](https://storybook.js.org/docs/svelte/get-started/install) for implementation.

[Planning Poker]: https://github.com/redfe/planning-poker-svelte
[Storybook]: https://storybook.js.org/
